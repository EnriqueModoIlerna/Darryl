# Darryl
Chatbot que aprende de conversaciones pasadas y adapta su comportamiento a cada usuario.

## Descargas necesarias
Ir a https://ollama.com/download

Una vez descargado en una terminal CMD instalar / correr ``phi3``, el modelo IA
C:\Users\contactediazperez>ollama run phi3

## Para lanzar el proyecto
Dentro de la carpeta del proyecto, primero en un CMD nuevo externo lanzar:

ollama run phi3

Posteriormente lanzar: 

python -m app.main
